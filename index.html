    Our Weather

    O projeto Our Weather, teve como temática os Objetivos de Desenvolvimento Sustentável(ODS), onde foi escolhido a ODS 13: Ação contra a mudança global do clima. Para realizar o desenvolvimento do projeto utilizei a linguagem de marcação HTML e para estilização o CSS. Neste projeto tive como principal desafio inovar no layout, consegui ultrapassar esse desafio utilizando a tag Flex do CSS. Assim, atingi o resultado esperado! Um design fluído e totalmente responsivo.

    O projeto Bistrô Quente, teve como objetivo a criação de um cardápio. Com isso, criei o projeto pensando em um restaurante completo. Apresentando: página sobre nós, delivery e cardápio.
